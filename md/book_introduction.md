# 内容简介

《基于 Android 平台的移动应用开发》v2 版面向希望系统学习现代 Android 开发的读者。与旧版相比，本版不再把现代实践作为零散补充，而是直接以当前主流工程方法为主线组织内容：Kotlin、AndroidX、ViewModel、Flow、Room、RecyclerView、WorkManager、Activity Result API、MediaStore 与 Photo Picker 等能力将贯穿全书。

本书仍然保留“概念讲解 + 结构化示例 + 综合实践”的教学方式，但在章节编排上进行了重构。新版先建立项目、构建和组件生命周期的基础认知，再展开界面、导航、交互、网络、存储和后台任务等主题，最后通过综合案例把这些能力串联起来，帮助读者从“会写单个示例”过渡到“能组织一个可维护的 Android 应用”。

从学习路径看，全书分为四个层次：

1. 基础层：第 1 至第 3 章，建立平台认知、项目结构认知和组件生命周期认知。
2. 界面层：第 4 至第 7 章，聚焦布局、资源、控件、导航、结果回调和跨应用交互。
3. 数据层：第 8 至第 10 章，覆盖远程数据获取、本地存储、文件访问与本地数据层设计。
4. 系统能力层：第 11 章，组织异步、后台任务、通知、定位与综合案例。

本书适合以下读者：

- 具备基础编程经验，希望系统学习 Android 的初学者。
- 已经接触过旧版 Android API，但希望迁移到现代实践的读者。
- 需要将教材内容用于课堂教学、培训或内部知识整理的教师与工程师。

阅读建议如下：

- 先按目录顺序完整通读一次，建立整体知识图谱。
- 学每章时至少完成一个“最小可运行任务”，不要只停留在 API 理解上。
- 阅读历史 API 或迁移说明时，要明确区分“仍会见到”和“新项目应采用”的方案。
