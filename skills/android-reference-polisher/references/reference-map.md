## Selection hints

- `Kotlin`, `Flow`, `Compose`, `ViewModel`, `Jetpack`:
  Start with `Mastering Kotlin for Android 14` or `Kickstart Modern Android Development With Jetpack And Kotlin`.

- `project`, `case study`, `practice`, `mini app`:
  Start with `Tiny Android Projects Using Kotlin 2024`.

- `chapter tone feels like documentation`, `needs smoother prose`, `needs broader development framing`:
  Start with `Thriving in Android Development Using Kotlin`.

- If a chapter needs both modern practice and a more teachable example path:
  Pair one of the Jetpack/Kotlin books with `Tiny Android Projects Using Kotlin 2024`.

## Use policy

- Prefer 1 or 2 books per rewrite.
- Add a third only if the first two clearly do not cover a teaching gap.
- Do not open every reference for every chapter.
